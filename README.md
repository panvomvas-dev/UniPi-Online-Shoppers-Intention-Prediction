# UniPi-Online-Shoppers-Intention-Prediction
Academic Data Science project (UniPi) on the 'Online Shoppers Purchasing Intention Dataset'. Built ML pipelines (Scikit-Learn &amp; PySpark) to predict purchase revenue (12k+ sessions). Applied Preprocessing, Clustering (K-Means/DBSCAN), and Classification (RF/SVM/NB) to gain insights. Delivered a full technical report on e-commerce behavior.
